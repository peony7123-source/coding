Chapter 02. 입출력과 자료

1. 화면에 출력하기: print()

학습 포인트

print는 괄호 안의 메시지(인수)를 화면에 보여주는 명령어입니다.

print("hello world")
hello world
💡 예약어 (Keyword)
- print와 같이 특정한 의미가 약속된 단어를 예약어라고 합니다.
- 특별한 의미가 있어 미리 역할이 정해져 있는 단어입니다.
- 다른 목적으로 사용할 수 없으며 print, input, if, while, for 등이 있습니다.

1) 문자열 출력하기-1

학습 포인트

출력할 문자열을 큰 따옴표(") 또는 작은 따옴표(')로 감싸서 사용합니다.

1. 문자열 출력하기

print("Hello world~!!!") print('Hi~ Python')
Hello world~!!!
Hi~ Python

2. 에러가 발생하는 경우

⚠️ 주의! 따옴표 없이 글자만 입력하면 에러가 발생해요.

print(hello world)
SyntaxError: invalid syntax

3. 둘 이상의 인수를 넣을 경우

여러 개의 데이터를 한 번에 출력하고 싶을 때는 쉼표(,)로 구분한다는 점을 기억하세요.

print("hello", "world")
hello world

4. 빈 print()의 역할

- print() 처럼 괄호 안에 아무런 내용을 넣지 않으면 '한 줄 띄우기' 역할을 합니다.
- 내용 없이 줄바꿈만 수행하여 출력물 사이의 간격을 조절할 때 유용합니다.

print("hello world") print() print("hi python")
hello world

hi python
❓ 인수(Argument)란?
- 함수의 괄호 안에 들어가는 값(쉼표로 구분되는 값)을 말합니다.

  • (2) 2개 인수를 사용하여 "오늘은 즐거워"를 출력하세요.

    • 2) 작은 따옴표 또는 큰 따옴표 출력하기

    학습 포인트

    문장 안에 따옴표를 넣고 싶을 때는 서로 다른 종류의 따옴표로 감싸면 됩니다.

    1. 큰 따옴표(") 출력할 때

    print(' "Hello world~!!!" ')
    "Hello world~!!!"

    2. 작은 따옴표(') 출력할 때

    print(" 'Hello world~!!!' ")
    'Hello world~!!!'
    코드 작성 실습
  • (1) 나는 '파이썬'을 좋아해
  • (2) 만나면 "안녕"이라고 인사하기
    • 📌 참고: 이스케이프 코드(\' 또는 \") 사용

    따옴표 앞에 역슬래시(\)를 붙여도 문자로 인식됩니다.

    print("Python\'s favorite subject.") print("\"나는 파이썬을 좋아해\"")
    Python's favorite subject.
    "나는 파이썬을 좋아해"

    3) 여러 줄 출력과 문장 구분하기

    학습 포인트

    문장을 구분하거나 여러 줄의 문자열을 한 번에 다루는 방법입니다.

    1. 한 줄에 여러 명령 쓰기 : 세미콜론(;)

    파이썬은 원칙적으로 한 줄에 하나의 명령을 쓰지만, 세미콜론(;)을 사용하면 여러 명령을 한 줄에 나열할 수 있습니다.

    # 세미콜론으로 구분하여 한 줄에 작성 print("코");print("딩");print("하")
    코
    딩
    하

    2. 여러 줄 문자열 출력하기 : 따옴표 3개(""" 또는 ''')

    큰따옴표나 작은따옴표를 3개 연속으로 쓰면 문장 안에서 줄 바꿈을 자유롭게 할 수 있습니다.

    print("""코 딩 파 이 썬""")
    코
    딩
    파
    이
    썬
    ⚠️ 주의사항
    일반적인 따옴표 하나(")만 사용한 상태에서 코드를 줄 바꿈 하면 에러가 발생합니다.
    # 에러가 발생하는 예시 print("코 딩")

    4) 숫자 출력하기

    학습 포인트

    숫자는 따옴표 없이 입력하며, 컴퓨터가 바로 계산을 수행할 수 있습니다.

    연산자설명예시
    +더하기2 + 3 = 5
    -빼기5 - 2 = 3
    *곱하기3 * 4 = 12
    /나누기 (실수)5 / 2 = 2.5
    //몫 (정수)5 // 2 = 2
    %나머지5 % 2 = 1
    **거듭제곱2 ** 3 = 8
    ⚠️ 나누기(/) 연산 주의사항
    나누기(/)의 결과는 항상 실수(소수점이 있는 수)로 나옵니다.
    예를 들어 6 / 3 같이 딱 나누어떨어져서 소수점이 0이어도 결과는 2.0이 됩니다.

    • 6) 공백 출력하기

    학습 포인트

    파이썬은 쉼표와 문자열 내부의 공간을 통해 공백을 조절할 수 있습니다.

    1. 문자열 안의 공백은 그대로 출력됩니다.

    print("파 이 썬")
    파 이 썬

    2. 쉼표로 구분된 인수는 기본적으로 한 칸의 공백을 만듭니다.

    print("파","이","썬")
    파 이 썬

    3. 쉼표 뒤의 물리적 공백은 출력 결과에 영향을 주지 않습니다.

    print("파", "이", "썬") print("파", "이", "썬")
    파 이 썬
    파 이 썬

    4. 문자열 내부 공백과 인수 공백이 만나면 간격이 넓어집니다.

    print("파 ", "이 ", "썬 ")
    파 이 썬

    7) 이스케이프 코드

    학습 포인트

    특수한 역할을 하는 문자 조합입니다.
    역슬래시(\)는 키보드의 ₩ 키를 누르면 입력됩니다.

    1. \n : 다음 줄로 줄을 바꿀 때 사용합니다. (Enter 역할)

    print("하이\n파이썬")
    하이
    파이썬

    2. \t : 탭(tab)만큼 간격을 띄울 때 사용합니다.

    print("1234567890") print("하이\t파이썬")
    1234567890
    하이 파이썬

    3. \\ : 역슬래시 문자 자체를 출력하고 싶을 때 사용합니다.

    print("하이\\파이썬")
    하이\파이썬

    4. 활용 예시 : 메뉴판처럼 정렬할 때 아주 유용해요!

    print("-"*30) print("아메리카노\t2000원") print("카푸치노\t5000원") print("라떼\t\t2000원") print("-"*30)
    ------------------------------
    아메리카노 2000원
    카푸치노 5000원
    라떼 2000원
    ------------------------------

    8) 인수 구분자 설정하기

    학습 포인트

    sep과 end 옵션을 사용하면 출력물의 사이와 끝을 내 마음대로 조절할 수 있습니다.

    8-1) sep 옵션

    sep(separator)은 인수를 구분하는 문자를 설정합니다. 기본값은 공백 하나(' ')입니다.

    print("naver", "com", sep=".") print("010", "1234", "5678", sep="-") print("python", "java", "c++", sep=" vs ") print("하이", "파이썬", sep="")
    naver.com
    010-1234-5678
    python vs java vs c++
    하이파이썬

    8-2) end 옵션

    end는 출력을 마친 후 끝에 붙일 문자를 설정합니다. 기본값은 줄바꿈 문자('\n')입니다.

    print("Hello", end=" ") print("World") print("Python", end="!")
    Hello World
    Python!
