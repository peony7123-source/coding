Chapter 02. Basic

1. 출력 print()

✅ print는 괄호 안의 메시지(인수)를 출력하는 명령어입니다.

Python Code
# print는 괄호안의 메시지(인수)를 출력하는 명령어이다. print("hello world")
hello world
Python Code
# 둘 이상의 인수를 넣을 경우, 쉼표(,)로 구분한다. print("hello", "world")
hello world
💡 예약어 (Keyword)
- print와 같이 특정한 의미가 약속된 단어를 예약어라고 합니다.
- 특별한 의미가 있어 미리 역할이 정해져 있는 단어입니다.
- 다른 목적으로 사용할 수 없으며 print, input, if, while, for 등이 있습니다.

1) 문자열 출력하기-1

출력할 문자열을 큰 따옴표(") 또는 작은 따옴표(')로 묶어줍니다.

print("Hello world~!!!") print('Hi~ Python')
Hello world~!!!
Hi~ Python

⚠️ 따옴표 없이 출력할 메시지를 입력하면 에러가 발생합니다.

print(hello world)
SyntaxError: invalid syntax

    5) 공백 출력하기

    ❓ 인수(Argument)란?
    - 함수의 괄호 안에 들어가는 값(쉼표로 구분되는 값)을 말합니다.

    1. 문자열 안의 공백은 그대로 출력됩니다.

    print("파 이 썬")

    2. 쉼표로 구분된 인수는 기본적으로 한 칸의 공백을 만듭니다.

    print("파","이","썬")

    3. 쉼표 뒤의 물리적 공백은 출력 결과에 영향을 주지 않습니다.

    print("파", "이", "썬") print("파", "이", "썬")

    4. 문자열 내부 공백과 인수 공백이 만나면 간격이 넓어집니다.

    print("파 ", "이 ", "썬 ")

    6) 이스케이프 코드

    ✅ 프로그래밍할 때 사용할 수 있도록 미리 정의해 둔 문자 조합입니다. 주로 출력문을 보기 좋게 정렬하기 위해 사용합니다. 역슬래시(\)는 키보드의 ₩ 키입니다.

    1. \n : 다음 줄(new line)로 줄을 바꿉니다.

    print("하이\n파이썬")

    2. \t : 탭(tab) 문자로 8칸마다 공백을 채웁니다.

    print("하이\t파이썬")

    3. \\ : 역슬래시(\) 문자 자체를 출력합니다.

    print("하이\\파이썬")

    4. 실제 활용 예시 : 메뉴판처럼 정렬된 출력이 가능합니다.

    print("아메리카노\t2500원") print("카페라떼\t3000원")
    아메리카노 2500원
    카페라떼 3000원

    7) 인수 구분자 설정하기

    7-1) sep 옵션

    ✅ sep은 'separator'의 약자로, 인수 사이의 구분자를 지정합니다. (기본값: 공백 " ")

    print("naver", "com", sep=".") print("010", "1234", "5678", sep="-") print("python", "java", "c++", sep=" vs ") print("하이", "파이썬", sep="")
    naver.com
    010-1234-5678
    python vs java vs c++
    하이파이썬

    7-2) end 옵션

    ✅ end는 출력 마지막에 무엇을 붙일지 지정합니다. (기본값: 줄바꿈 \n)

    print("Hello", end=" ") print("World") print("Python", end="!")
    Hello World
    Python!
